Cara menggunakan aplikasi ValidasiOrder

Alur Kerja

1
Setup Data Master — Upload All Orderan historis + All RTS ke menu Data Master → Simpan ke Database.
2
Validasi Harian — Upload file orderan baru hari ini → Jalankan Validasi.
3
Baca Hasil — Sistem cek HP vs data master. Duplikat & RTS otomatis terdeteksi.
4
Update Master — Upload ulang All Orderan berkala agar data selalu terkini.

Warna Highlight Baris

Biru — Duplikat Team (HP sama di file hari ini)
Merah — Duplikat All Team (HP pernah order di data master)
Kuning — Pernah RTS (termasuk yang duplikat + RTS)

Warna Kolom Ekspedisi

Normal — ekspedisi sesuai rekomendasi ✓
Merah — ekspedisi tidak ada di daftar rekomendasi ⚠

Warna Kolom ACC SPV CS

Hijau — KIRIM
Kuning — HOLD
Merah — CANCEL

Warna Kolom Wilayah Rawan

Merah — ≥30% RTS (rawan tinggi)
Oranye — 15–29% RTS (perlu diperhatikan)
Hijau — <15% RTS (relatif aman)
