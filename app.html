Panduan Penggunaan

1. Login

Gunakan akun email yang sudah didaftarkan oleh admin di Supabase Authentication.

2. Import Data

Export dari Google Sheets sebagai CSV atau XLSX, lalu upload ke kotak import masing-masing. Isi kolom range jika ingin membatasi baris yang diproses (opsional).

3. Jalankan Validasi

Klik "Jalankan Validasi". Sistem akan mendeteksi duplikat berdasarkan nomor HP yang sama, dan mencari riwayat RTS berdasarkan RESI di semua kolom bulan (Jan–Des).

4. Warna Highlight

Merah muda — Duplikat (nomor HP sama dengan order lain)
Kuning — Pernah RTS (resi ditemukan di data RTS)

5. Export Hasil

Klik Export CSV atau Export XLSX. File akan berisi kolom tambahan: Status_Validasi, Is_Duplikat, RTS_Bulan, RTS_Reason, RTS_Status, RTS_Pihak.

6. Tambah Anggota Tim

Masuk ke Supabase Dashboard → Authentication → Users → Invite User. Masukkan email anggota tim baru.
